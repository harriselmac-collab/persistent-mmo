# Assets — Static Game Media

This directory contains static, optimized, and raw visual assets for the web interface.

## Folders
*   `items/`: High-definition icons for resources, tools, food, weapons, and manufactured items.
*   `regions/`: SVG vector maps and flags for countries and territories.
*   `avatars/`: Core player portrait frames, rank insignias, and organization logos.
*   `ui/`: System-wide gradients, background images, and custom animations.
