# Scripts — Operational & Seeding Tools

This directory contains executable scripts for database administration, validation, seed data imports, and local testing configurations.

## Contents
*   `db-seed-generator.js`: Programmatic generation of complex baseline items, formulas, and geographic region nodes.
*   `lint-schemas.sh`: Quality checker for database migration script conventions.
*   `deploy.sh`: Local convenience script wrapper for deploying edge functions and database schemas.
