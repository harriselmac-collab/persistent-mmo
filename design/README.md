# Design Guidelines & System Layouts

This directory contains design assets, wireframes, style guides, and documentation on the visual system of the MMORPG.

## Design Tokens
*   **Colors**: Custom HSL palettes (Dark slate foundations, neon highlights representing political power or war status).
*   **Typography**: Outfit or Inter font pairings for telemetry, tables, and statistics.
*   **Aesthetics**: Glassmorphism, smooth micro-animations on interactive map nodes, and dynamic hover indicators.

## Contents
*   `wireframes/`: Interactive component designs and page layouts.
*   `tokens.json`: Theme colors, gradients, and font size configuration maps for Tailwind integrations.
*   `assets/`: Vector icons, banner layouts, and dashboard layouts.
